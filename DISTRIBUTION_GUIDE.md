## 📋 ユーザー向けインストール手順

### システム要件
- **OS**: Windows 10/11（64bit）
- **必要な権限**: 管理者権限（初回インストール時のみ）
- **ディスク容量**: 約50MB

### インストール手順

1. **インストーラーを実行**
   
   HEIC_Converter_Setup_v1.0.0.exe をダブルクリック
   
2. **セキュリティ警告が表示される場合**
   - Windows Defender SmartScreenの警告が出る場合があります
   - 「詳細情報」→「実行」をクリック

3. **インストールウィザードに従う**
   - 言語: 日本語を選択
   - インストール先: デフォルトのまま推奨
   - デスクトップアイコン: お好みで選択
   - インストール実行

4. **完了**
   - 「HEIC Converterを起動」にチェックを入れて「完了」をクリック

## 🎯 アプリケーションの使い方

### 使い方

1. **HEICファイルをドラッグ&ドロップ**
   - ファイルまたはフォルダを画面にドロップ
   - 複数ファイル同時対応

2. **出力形式を選択**
   - PNG（透明度保持）
   - JPG（ファイルサイズ小）

3. **保存先を指定（オプション）**
   - 空白の場合: 一時プレビューのみ
   - 指定した場合: そのフォルダに保存

4. **変換実行**
   - 「変換」ボタンをクリック
   - 進捗バーで進行状況を確認

5. **結果を取り出し**
   - プレビュー画面に変換後の画像が表示
   - チェックを入れて複数選択
   - デスクトップやフォルダにドラッグ&ドロップで取り出し

## 🔧 アンインストール方法

1. Windowsの設定から「アプリ」→「インストールされているアプリ」
2. 「HEIC Converter」を探して「アンインストール」をクリック

## 🐛 トラブルシューティング

### インストール中にエラーが出る
- **原因**: 管理者権限がない
- **解決**: インストーラーを右クリック→「管理者として実行」

### アプリが起動しない
- **原因**: ウイルス対策ソフトがブロック
- **解決**: セキュリティソフトの例外設定に追加
